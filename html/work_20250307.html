안녕하세요. 강성민입니다.

수유국사(v2024122401 최신 펌웨어 배포완료)는 금일 새벽 장비작업을 수행 하였지만 해당 이슈는 발생하지 않았다고 합니다.

테스트 STB에 CCM reset 전송 후 히든메뉴에서 CM IP 사라졌다가 다시 IP가 잡히는 것을 확인 후 "cmts로 ping 시험" 성공을 확인 하였습니다. HOST IP를 못 받아 네트워크가 안되는 증상은 발생하지 않았습니다.

증상 발생시 soft reset(STB만 reset) 시 회복이 안되고 hard reset(CM & STB 모두 reset) 시 복구가 된다는 것은 이전에 발생한 블랙화면과 동일하게 CM 과 STB간의 인터페이스에 문제가 생겨서 stb reset만으로는 해결이 안되는 것 같습니다.
