https://jira.kaonmedia.com/browse/K0644-381

리커버리 모드에서 IR 무응답 간헐 발생 – user 빌드 한정 타이밍 이슈 개선
현상 : 리커버리 진입 시 간헐적으로 입력 무응답. userdebug 빌드에서는 재현 안 됨,
user 빌드에서만 재현. (10회 중 1~2회 발생)
원인 : 리커버리 초기화 시점에 /dev/input/event*가 아직 준비되기 전에 초기 스캔이 먼저 실행되어 초기화 시점 불일치(타이밍 문제) 로 장치 인식을 놓침.
userdebug 대비 user 빌드가 더 빠르게 동작해 타이밍 불일치가 심화됨.
수정 : minui/events.cpp::ev_init() 초입에 user 빌드에서만 300ms 지연을 추가
