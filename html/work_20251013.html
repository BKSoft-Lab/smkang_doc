이미지 검증 : Input 폴더에 있는 img 파일들을 이용하여 각 파티션에 flash 하여 gang image 확인

1. 부팅 확인한 시료 준비
2. OQC S/W를 USB에 복사
3. USB를 시료에 장착 후 전원 인가
4. quit 입력하여 OQC S/W 종료
5. USB 제거 준비
umount /mnt/usb
6. USB 제거
7. output 폴더내 하기 파일들을 ext4 F/S로 포맷된 USB에 복사 (userdata.bin 용량 문제로 FAT32 사용 불가)

9. 복사한 파일들을 eMMC에 flash
1) MAIN F/W

file=/mnt/ext4/boot_a.bin
