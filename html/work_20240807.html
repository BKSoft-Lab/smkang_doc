https://jira.kaonmedia.com/browse/K1023-182

Tombstone 에러 발생 재현테스트
1. 주파수를 감쇄 필터 42(15+15+12) 사용 후 테스트
- LIVE 채널 시청 에이징 : 발생 안함
- VOD 재생 에이징 : 네트워크 단절로 VOD 재생불가
2. widevine이 외부 접속하는 443 포트 차단 후 VOD 재생 테스트
- 차단 명령어
su
iptables -A OUTPUT -p tcp --dport 443 -j DROP
iptables -A OUTPUT -p udp --dport 443 -j DROP
- 차단해제 명령어
iptables -D OUTPUT -p tcp --dport 443 -j DROP
iptables -D OUTPUT -p udp --dport 443 -j DROP
