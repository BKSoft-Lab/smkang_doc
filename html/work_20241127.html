Node.js 설치
NVM 설치:
NVM(Node Version Manager)을 사용하면 쉽게 Node.js의 여러 버전을 관리할 수 있습니다.
다음 명령어를 사용하여 NVM을 설치합니다:
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
설치가 완료되면, 다음 명령어로 현재 셸 세션에 NVM을 로드합니다:
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
NVM이 정상적으로 설치되었는지 확인하기 위해 다음 명령어를 실행합니다:
command -v nvm
nvm이라고 출력되면 성공적으로 설치된 것입니다.
NVM을 사용하여 Node.js 18.20.5를 설치합니다:
nvm install 18.20.5
버전 확인:
node -v
Node.js 버전 18.20.5가 출력되어야 합니다.
