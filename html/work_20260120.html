3단계: 팀원 초대하기 (BKSoft 멤버 추가)

원들이 이 프로젝트에 접근할 수 있도록 권한을 줍니다.
저장소 상단 메뉴의 [Settings] → 왼쪽 사이드바 **[Collaborators]**를 클릭합니다.
[Add people] 버튼을 누릅니다.
팀원의 GitHub ID나 이메일을 검색하여 선택한 후 **[Add to this repository]**를 클릭합니다.
팀원에게 이메일로 초대장이 발송됩니다. 팀원이 이메일에서 초대를 수락해야 최종적으로 참여가 완료됩니다.

4단계: 비주얼 스튜디오와 연결하기

이제 각자의 컴퓨터에 있는 비주얼 스튜디오에서 코드를 올리고 내려받을 차례입니다.
비주얼 스튜디오 실행: 작업 중인 프로젝트를 엽니다.
Git 초기화: 하단 상태 표시줄의 [Git에 추가] 또는 메뉴에서 **[Git] -> [Git 저장소 만들기...]**를 선택합니다.
원격 서버 연결: * 기존 원격을 선택합니다.
원격 URL: GitHub 저장소 페이지의 [<> Code] 버튼을 눌러 나오는 주소(https://github.com/BKSoft-Lab/AutoTest.git)를 복사해 붙여넣습니다.
푸시(Push): **[만들기 및 푸시]**를 누르면 내 코드가 GitHub 서버로 전송됩니다.

5단계: 매일 사용하는 협업 사이클

팀원들과 일할 때는 다음 세 가지만 기억하세요.
1. pull : 가져오기 (서버의 최신 코드를 내 PC로 받기 / 출근 직후, 작업 시작 전)
2. commit : 저장하기 (내가 수정한 내용을 내 PC에 기록 / 의미 있는 코드 수정을 마쳤을 때)
3. push : 올리기 (내 PC의 기록들을 서버에 공유 / 하루 일과 종료 전, 기능 완성 후)
초보자 팁: 코드를 수정하기 전에 반드시 [Pull]을 먼저 해서 다른 사람이 올린 내용이 있는지 확인하세요.
그래야 나중에 코드가 꼬이는 '충돌'을 줄일 수 있습니다.

5. [복제] 버튼을 누릅니다.
6. 완료 후 우측하단에 브랜치 master 선택
