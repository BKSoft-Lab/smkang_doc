build script 이용 Build 방법
1) ./k1200ua_build.sh -c
: u-boot, kernel, android clean
2) ./k1200ua_build.sh –m
: android oem image build
3) ./k1200ua_build.sh -o
: include otapackage image
4) ./k1200ua_build.sh -s
: only android build (u-boot, kernel build skip)
5) ./k1200ua_build.sh -u
: user mode build
